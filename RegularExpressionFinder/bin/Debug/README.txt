//   Regular Expression Finder     //

■はじめに

本ツールは、膨大なテキストファイル(ログファイル等)から、
複数の正規表現(AND条件/OR条)で検索し、マッチする行だけを抽出するツールです。

本ツールの動作には、.net Framework 4.5以降と、
本ツールを動かすフォルダに対する書き込み権限が必要です。



■使い方

1.filesフォルダに、検索対象となるテキストファイルを全部入れます。拡張子は何でもいいです。

2.expressions.txtに、検索する正規表現を記入して保存します。
  検索対象１行に対して複数の正規表現をOR条件で指定する場合は、条件を１行つずつ記入します。
  検索対象１行に対して複数の正規表現をAND条件で指定したい場合は、それらを&で繋ぎます(半角のアンド)。
  特定の名前を含むファイル名のときのみ検索条件とする場合は、
  検索条件の後ろに【server1】のように墨付きかっこで指定します。
  
3.RegularExpressionFinder.exeを起動し、「3.検索を開始」を押します。

4.検索した結果が「answer_(現在日時).txt」として以下のように出力されます。

----------------出力例------------------------------------------------------------------
[ファイル名]May 29 21:54:14 fumidai kdumpctl: Starting kdump: [FAILED]
[ファイル名]May 29 21:54:14 fumidai systemd: Failed to start Crash recovery kernel arming.
[ファイル名]May 29 21:54:14 fumidai systemd: Unit kdump.service entered failed state.
[ファイル名]May 29 21:54:14 fumidai systemd: kdump.service failed.
